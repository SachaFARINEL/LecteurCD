
Pour lancer le projet, il faut effectuer les lignes de commandes suivantes :
    Pour la compilation :
        - javac -d ../class ../src/ihm/FrameLecteurCD.java ../src/control/MyMouseListener.java ../src/datas/*.java
    Pour le lanceur :
        - java ihm.FrameLecteurCD
